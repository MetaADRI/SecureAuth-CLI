# SecureAuth Node.js - Quick Start Guide

## 🚀 Get Running in 3 Steps

### Step 1: Install Dependencies
```bash
npm install
```

### Step 2: Start Server
```bash
npm start
```

### Step 3: Open Browser
```
http://localhost:3000
```

---

## 📋 What You Get

✅ **Backend:** Fully complete (all files created)
✅ **Frontend:** Copy from Phase 3 ZIP (HTML/CSS/JS files)

The frontend files are the SAME as the Python version since they're just HTML/CSS/JavaScript (framework-agnostic).

---

## 📦 Frontend Setup (One-Time)

Copy these files from the Phase 3 frontend ZIP to the `public/` folder:

```
From Phase 3 ZIP → To Node.js Project:

phase3-frontend/index.html       → public/index.html
phase3-frontend/register.html    → public/register.html  
phase3-frontend/login.html       → public/login.html
phase3-frontend/verify.html      → public/verify.html
phase3-frontend/dashboard.html   → public/dashboard.html
phase3-frontend/css/style.css    → public/css/style.css
phase3-frontend/js/auth.js       → public/js/auth.js
phase3-frontend/js/inactivity.js → public/js/inactivity.js
```

**OR** use the ones in this package (already included in the ZIP).
